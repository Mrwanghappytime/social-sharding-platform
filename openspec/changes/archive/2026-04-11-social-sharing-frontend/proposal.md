## 为什么

后端微服务已完成，需要开发 Vue 前端应用与后端 API 集成，实现用户注册登录、动态发布、互动评论、关注关系和实时通知等功能。

## 变更内容

创建完整的前端单页应用（SPA），包括：

1. **用户认证模块** - 注册、登录、JWT Token 管理
2. **动态模块** - 发布、查看、删除动态，支持图片/视频
3. **互动模块** - 评论、点赞功能
4. **社交关系模块** - 关注/取消关注、粉丝列表
5. **通知模块** - 实时通知推送（WebSocket）、通知列表
6. **文件上传模块** - 图片/视频上传预览
7. **搜索模块** - 动态标题搜索

## 功能 (Capabilities)

### 新增功能

- `user-auth`: 用户注册、登录、Token 管理、个人信息展示
- `post-feed`: 动态流展示、发布动态、删除动态
- `media-upload`: 图片/视频上传组件，支持拖拽和预览
- `interaction`: 评论、点赞功能及状态展示
- `social-graph`: 关注列表、粉丝列表、关注操作
- `notification`: 实时通知、通知列表、已读标记
- `search`: 动态搜索功能

## 影响

- 新增 `frontend/` 目录，包含 Vue 3 + Vite 项目
- 依赖后端 Gateway API (端口 8080)
- 使用 WebSocket 连接通知服务
- 集成 Nginx 部署配置
