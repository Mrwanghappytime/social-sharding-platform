## 新增需求

### 需求:按标题搜索动态
系统必须支持通过标题关键词搜索动态，使用 MySQL LIKE 查询。

#### 场景:搜索结果存在
- **当** 用户使用关键词搜索动态
- **那么** 系统返回标题包含关键词的动态列表

#### 场景:搜索结果为空
- **当** 用户搜索一个不存在的关键词
- **那么** 系统返回空列表

#### 场景:搜索关键词为空
- **当** 用户提交空关键词搜索
- **那么** 系统返回错误码 400 Bad Request

#### 场景:搜索结果分页
- **当** 用户搜索动态并指定分页参数
- **那么** 系统返回对应分页的搜索结果
